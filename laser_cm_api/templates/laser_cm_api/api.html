API Documentation

This API documentation provides information about the available endpoints and their usage.

LaserMarkingParametersList

Endpoint: GET /api/color-parameters/

Description: Retrieves a list of color parameters based on RGB values.

Usage: Use this endpoint to retrieve a list of color parameters that match the provided RGB value.

Query Parameter: hex_color (string): The RGB color value in hexadecimal format (e.g., "#RRGGBB").

Response: Returns a list of color parameters that match the provided RGB value.

Curl Request:

curl -X GET http://your-api-domain/api/color-parameters/?hex_color=#RRGGBB

LaserMarkingParametersDetail

Endpoint: GET /api/color-parameters/{id}/

Description: Retrieves details of a specific color parameter.

Usage: Use this endpoint to retrieve detailed information about a color parameter based on its ID.

URL Parameter: id (integer): The ID of the color parameter.

Response: Returns detailed information about the specified color parameter.

Curl Request:

curl -X GET http://your-api-domain/api/color-parameters/{id}/

ColorSearchAPIView

Endpoint: POST /api/search-color/

Description: Search for color parameters based on RGB values.

Usage: Use this endpoint to search for color parameters based on an RGB value. The API will return matching colors from the database.

Request Body:

{
    "hex_color": "#RRGGBB"
}

Response:

{
    "matching_colors": [
        {
            "color_red": 255,
            "color_green": 0,
            "color_blue": 0,
            // Other LaserMarkingParameters fields...
        },
        // Other matching colors...
    ]
}

Curl Request:

curl -X POST -H "Content-Type: application/json" -d '{"hex_color": "#RRGGBB"}' http://your-api-domain/api/search-color/
